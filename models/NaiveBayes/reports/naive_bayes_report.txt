VARIANTES TESTADAS
--------------------------------------------------------------------------------
1. GaussianNB:
   - Assume distribuição Gaussiana (Normal) para features contínuas
   - Melhor para features com valores reais
   - Não requer normalização

2. MultinomialNB:
   - Projetado para features de contagem/frequência
   - Requer features não-negativas
   - Features escaladas [0, 100] para simular contagens
   - Alpha=1.0 (Laplace smoothing)

3. BernoulliNB:
   - Projetado para features binárias (0/1)
   - Features binarizadas usando mediana como threshold
   - Alpha=1.0 (Laplace smoothing)
